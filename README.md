## 📖 Sobre o Projeto

O **AgroNutri** é um site institucional desenvolvido em **HTML5** e **CSS3**, com um design moderno e responsivo, voltado para a divulgação de informações sobre alimentação animal.

O projeto foi criado com foco em apresentar conteúdos de forma clara e organizada para produtores rurais, estudantes, veterinários e demais interessados em nutrição animal.

---

## 🎯 Objetivos

- Informar sobre alimentação animal.
- Apresentar boas práticas nutricionais.
- Demonstrar um layout profissional.
- Servir como projeto de estudo para HTML e CSS.

---
